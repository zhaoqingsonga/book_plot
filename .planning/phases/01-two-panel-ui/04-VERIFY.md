# VERIFY.md — Phase 1

## 验证标准
1. 启动 app.R 后，"田间记录管理" tab 显示两级面板（左侧试验列表 + 右侧空白提示"请选择试验"）
2. 左侧列表显示所有试验（按 experiment_id 分组），每项有试验名称、类型标签、记录数、地点
3. 输入关键词搜索，列表实时过滤
4. 选择类型下拉，列表按类型过滤
5. 点击一个试验，右侧显示该试验的所有田间记录
6. 点击田间记录行，弹出明细弹窗（显示约120个字段）
7. 点删除按钮，弹出确认框，确认后删除整个试验（所有田间记录）
8. 刷新后数据正确重新加载

cd ~/Documents/GitHub/melt/make_field_book && Rscript app.R
