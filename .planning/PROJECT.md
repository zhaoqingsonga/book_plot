# PROJECT.md — 试验管理两级视图

## 目标
将"田间记录管理"页面从单列表改造为两级视图：
- **左侧**：试验列表（按 experiment_id 分组，显示试验基本信息）
- **右侧**：选中试验的田间记录列表（支持查看、编辑、导出）

## 数据源
- `population_field_records` (200条)
- `line_selection_field_records` (56条)
- `yield_test_field_records` (1646条)
- 按 `experiment_id` 分组，每个试验显示：试验ID、试验名称、类型、记录数、地点、创建时间

## UI 设计

### 左侧面板（试验列表）
- 宽度：~350px
- 搜索框：按试验名称/ID/地点关键词筛选
- 筛选下拉：按类型（群体/株行/产比）
- 列表项（每个试验一行）：
  - 试验名称（主显示）
  - 类型标签 + 记录数 + 地点
  - 底部：创建时间
- 选中态：高亮背景色
- 操作：点击选中 + 显示删除按钮

### 右侧面板（田间记录列表）
- 宽度：剩余空间
- 显示选中试验的所有田间记录
- 列：fieldid(隐藏)、记录编号、阶段、地点、关键性状（收获株数、亩产、蛋白等）
- 支持：查看明细、导出
- 无选中试验时：显示空白占位提示

## 功能

### 左侧列表
- `filtered_experiments`: 从三个田间记录表按 experiment_id 分组聚合
- 搜索：模糊匹配试验名称、试验ID、地点
- 类型筛选：群体/株行/产比/全部
- 刷新版本计数器触发重渲染

### 右侧列表
- `field_records_for_experiment`: 根据选中的 experiment_id 查对应表的全部字段
- 查看：点击行弹出明细弹窗（所有性状列）
- 编辑：暂不实现（Phase 2）
- 导出：暂不实现（Phase 2）

### 删除
- 选中左侧试验项 → 显示删除按钮
- 点删除 → 确认弹窗 → 确认后删除该 experiment_id 下所有田间记录
- 从 population_field_records / line_selection_field_records / yield_test_field_records 同时删除
